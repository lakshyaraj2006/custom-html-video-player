# 🎬 Custom HTML Video Player

A modern **custom video player built with HTML, TailwindCSS, JavaScript, and Vite**.
It recreates many features of popular streaming players with a clean UI and smooth interactions.

---

## ✨ Features

* ▶ **Play / Pause**
* ⏱ **Seekbar with live progress**
* ⏳ **Current time / Remaining time toggle**
* 🔊 **Volume control with hover slider**
* 🔇 **Mute / Unmute**
* ⚡ **Playback speed control**
* 🔁 **Loop toggle**
* ⛶ **Fullscreen support**
* 💡 **Tooltips for all controls**
* 🎨 **Smooth Tailwind transitions**
* 🖱 **Hover-activated control bar**
* 📱 **Responsive aspect-ratio video container**

---

---

## 🛠 Tech Stack

* **HTML5 Video API**

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/lakshyaraj2006/custom-html-video-player.git
cd custom-html-video-player
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Start development server

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

The optimized build will be generated in:

```
dist/
```

Preview production build:

```bash
npm run preview
```
